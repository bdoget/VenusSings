# VenusSings
Interactive Karaoke Machine
### Setup
1. install miniconda
2. install react / npm
#### Setup Python Env
1. nav to env folder
2. conda env create -f venusSings.yml
3. Update env: conda env update --prefix ./env --file venusSings.yml  --prune
4. conda env export >> venusSings.yml
